# PC端常用浏览器的内核比较
  | 名称        | 内核          | 备注|
  | ----------  |:-------------:| -----:|
  | chrome   | WebKit | 暂无 |
  | firefox  | Gecko      | 暂无|
  | opera    | Presto     | 暂无 |
  | safari   | WebKit      | 暂无 |
  | IE       | IE6~IE11是Trident 内核    | 暂无 |
  |360浏览器 |Trident或者Blink（Webkit） |兼容模式(IE8)或者极速模式 |
  | QQ浏览器 |Trident或者Blink（Webkit） |兼容模式(IE8)或者极速模式 |
